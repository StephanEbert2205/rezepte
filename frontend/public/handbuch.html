Rezeptsammlung – Handbuch

Deine persönliche Rezeptsammlung – Rezepte aus dem Internet importieren, abfotografieren, teilen und gemeinsam kochen.

📋

Überblick

Die Rezeptsammlung ist eine private Web-App für Familien und Freunde. Alle Rezepte werden zentral gespeichert und sind auf jedem Gerät abrufbar.

FunktionBeschreibung
URL-ImportRezepte von Kochwebsites automatisch importieren
Foto-ImportRezepte aus Büchern oder Zetteln abfotografieren (KI-Erkennung)
Manuelle EingabeEigene Rezepte direkt eintippen
PortionsskalierungZutaten automatisch für jede Personenanzahl berechnen
KochmodusGroßschrift & Display bleibt aktiv
TeilenEinmalige Kopier-Links oder dauerhafte Freigaben
KontoverbindungenZwei Konten verknüpfen – beide sehen alle Rezepte
PWAAls App installierbar, funktioniert ohne Browser-Leiste
App-URL: https://rezepte.familie-ebert.net
🔑

Anmeldung

Die Rezeptsammlung nutzt ausschließlich Google OAuth zur Anmeldung – kein separates Passwort erforderlich.

  • Die App-URL im Browser öffnen: rezepte.familie-ebert.net
  • Auf Mit Google anmelden tippen
  • Das gewünschte Google-Konto auswählen
  • Fertig – die Sammlung wird sofort angezeigt
💡 Tipp: Immer dasselbe Google-Konto verwenden. Beim Wechsel des Kontos erscheint eine leere Sammlung.
📲

App installieren (PWA)

Die Rezeptsammlung kann als Progressive Web App (PWA) auf dem Home-Bildschirm installiert werden – sie verhält sich dann wie eine native App: kein Browser-Rahmen, schnellerer Start, direkt vom Home-Bildschirm öffenbar.

🤖 Android (Chrome, Edge, Samsung)

  1. App öffnen und anmelden
  2. Zum Profil wechseln (Foto oben rechts)
  3. Auf Installieren tippen – oder:
  4. Im Browser-Menü ⋮ → App installieren

🍎 iPhone / iPad (Safari)

  1. Die App in Safari öffnen (kein Chrome!)
  2. Auf das Teilen-Symbol tippen (unten in Safari)
  3. „Zum Home-Bildschirm" auswählen
  4. „Hinzufügen" bestätigen
💡 Nach einer Deinstallation kann Chrome mehrere Tage warten, bevor der automatische Install-Dialog wieder erscheint. In dieser Zeit: Browser-Menü → App installieren oder Zum Startbildschirm.

Teilen-Funktion (nur nach Installation)

Sobald die App installiert ist, erscheint „Rezeptsammlung" im System-Teilen-Dialog von Android Chrome. Ein Rezept aus dem Browser teilen → App öffnet sich → Import startet automatisch.

✅ Android Chrome ✅ Android Edge / Samsung ❌ iOS Safari (nicht unterstützt)
➕

Rezepte hinzufügen

Über Hinzufügen in der Navigationsleiste stehen drei Tabs zur Verfügung.

🌐 Per URL importieren

Rezept-URL einfügen und auf Importieren tippen. Die App erkennt automatisch: Titel, Beschreibung, Zutaten (mit Menge und Einheit), Zubereitungsschritte, Kochzeiten, Portionsanzahl, Tags, Nährwerte und Bild.

Unterstützte Quellen (Auswahl):

Chefkoch lecker.de GuteKüche BBC Good Food Allrecipes Essen & Trinken Küchengötter Alle Schema.org-Seiten
💡 Wenn eine Seite keine strukturierten Daten liefert (kein Schema.org), wird nur der Titel importiert. In diesem Fall lohnt sich der Foto-Import als Alternative.
📷 Rezept abfotografieren

Tab Foto wählen, ein Foto aufnehmen oder aus der Galerie auswählen, dann auf Rezept erkennen tippen. Die KI (Claude Vision) erkennt Titel, Zutaten und Zubereitungsschritte und füllt das Bearbeitungsformular automatisch aus. Danach prüfen und speichern.

Funktioniert mit:

  • Buchseiten und Zeitschriften
  • Handgeschriebenen Rezeptzetteln
  • Screenshots von Webseiten
  • Abfotografierten Rezeptkarten
💡 Gutes Licht und gerade Aufnahme verbessern die Erkennungsgenauigkeit deutlich. Das Ergebnis immer kurz überprüfen und ggf. korrigieren.
✏️ Manuell eingeben

Tab Manuell eingeben wählen. Alle Felder können direkt eingetippt werden:

  • Titel (Pflichtfeld), Beschreibung, Bild-URL
  • Portionen, Vorbereitungszeit, Kochzeit, Gesamtzeit
  • Diät-Flags (vegetarisch, vegan, glutenfrei, laktosefrei)
  • Tags (kommagetrennt)
  • Zutaten mit Menge, Einheit, optionalem Hinweis
  • Zubereitungsschritte
🔄 Direkt aus dem Browser importieren

Drei Methoden, ohne die App manuell zu öffnen:

MethodePlattformEinrichtung
PWA-Teilen Android (installierte App) Kein Setup – Teilen-Menü → Rezeptsammlung
iOS-Kurzbefehl iPhone / iPad Einmalig in der Kurzbefehle-App einrichten (→ Details)
HTTP Shortcuts Android (ohne App-Installation) App aus dem Play Store + Einrichtung (→ Details)

Suchen und Filtern

Die Startseite zeigt alle Rezepte – eigene und die verknüpfter Konten – in einer durchsuchbaren Übersicht.

Volltextsuche

Das Suchfeld durchforstet Titel, Beschreibung, Zutaten und Tags gleichzeitig. Die Ergebnisse aktualisieren sich bei jeder Eingabe sofort.

Filter

Filteroptionen lassen sich beliebig kombinieren:

  • Ernährungsweise: Vegetarisch, Vegan, Glutenfrei
  • Maximale Kochzeit: Rezepte die innerhalb X Minuten fertig sind
  • Tags: Nach beliebigen Tags filtern (z. B. „Pasta", „Schnell", „Backen")
Rezepte anderer Konten

Beim Import aus einem verknüpften Konto wird der Besitzername auf der Rezeptkarte angezeigt. Die Volltextsuche und alle Filter gelten für alle sichtbaren Rezepte gleichzeitig.

📖

Rezeptansicht

Ein Tipp auf eine Rezeptkarte öffnet die Detailansicht mit allen Informationen.

⚖️ Portionsskalierung

Der Schieberegler über den Zutaten passt alle Mengenangaben stufenlos an die gewünschte Personenzahl an. Die Berechnung erfolgt anteilig – ganze Zutaten ohne Mengenangabe (z. B. „Salz") bleiben unverändert.

👨‍🍳 Kochmodus

Der Kochmodus-Button (Auge-Icon) vergrößert die Schrift deutlich für eine bessere Lesbarkeit auf Distanz. Zusätzlich bleibt der Bildschirm aktiv – das Display schaltet sich nicht automatisch ab (sofern in den Einstellungen aktiviert).

💡 Den Kochmodus bei Tablet-Montage an der Küchenwand nutzen – kein Entsperren mehr nötig.
🔀 Original / Angepasste Zutaten

Wenn für ein Rezept eigene Mengen oder Zutaten hinterlegt wurden (im Bearbeitungsformular), erscheint ein Umschalter Original / Angepasst über der Zutatenliste. So lassen sich z. B. abweichende Mengen für die Lieblingsvariante speichern.

🥗 Nährwerte

Sofern die Originalquelle Nährwertangaben enthält, werden sie pro Portion angezeigt: Kalorien, Protein, Fett und Kohlenhydrate.

🔗 Originalquelle

Bei importierten Rezepten ist ein Link zur Originalseite verfügbar („Original-Rezept").

✏️

Rezept bearbeiten & löschen

Der Bearbeiten-Button ist in der Rezeptansicht sichtbar (nur für berechtigte Nutzer).

Stammdaten bearbeiten

Alle Grunddaten können geändert werden: Titel, Beschreibung, Bild-URL, Kochzeiten, Portionsanzahl und Diät-Flags (vegetarisch, vegan, glutenfrei, laktosefrei).

Tags

Tags werden als kommagetrennte Liste eingegeben. Bereits in der Sammlung verwendete Tags werden als Vorschläge angeboten.

Eigene Zutaten hinterlegen

Im Abschnitt „Eigene Zutaten" können alternative Mengen oder Zutaten eingetragen werden – z. B. für eine bevorzugte Variante oder abweichende Mengen. In der Rezeptansicht erscheint dann der Umschalter Original / Angepasst.

🗑️ Rezept löschen

Das Papierkorb-Symbol in der Rezeptansicht entfernt das Rezept nach einer Bestätigung dauerhaft. Achtung: Geteilte Rezepte werden für alle Beteiligten gelöscht.

⚠️ Bearbeitungsrecht haben nur der Besitzer des Rezepts sowie Nutzer, denen explizit Bearbeitungsrecht über die Freigabe-Funktion erteilt wurde.
🔗

Rezepte teilen

Über das Teilen-Symbol (🔗) in der Rezeptansicht gibt es zwei Möglichkeiten. Das Menü ist nur für den Besitzer des Rezepts sichtbar.

📋 Kopier-Link (einmaliger Link)

Erstellt einen Link, den beliebige Personen öffnen können – auch ohne Anmeldung. Der Empfänger sieht eine Vorschau des Rezepts und kann es mit einem Tipp in die eigene Sammlung kopieren. Danach sind die beiden Rezepte völlig unabhängig voneinander – Änderungen werden nicht synchronisiert.

💡 Ideal zum einmaligen Weitergeben an Freunde oder die Familie. Der Link ist unbegrenzt gültig.
👤 Direkte Freigabe (dauerhaft)

Teilt das Rezept dauerhaft mit einem anderen registrierten Nutzer. Das Rezept bleibt ein einziges – beide sehen immer die aktuelle Version.

Berechtigungen:

  • Nur ansehen: Der andere Nutzer kann das Rezept lesen, aber nicht ändern.
  • Ansehen & bearbeiten: Änderungen beider Nutzer sind sofort für alle sichtbar.

Bestehende Freigaben werden im selben Dialog angezeigt und können dort entfernt werden. Die E-Mail-Adresse des Empfängers muss registriert sein.

👥

Konten verknüpfen

Unter Profil (Tipp auf das Profilfoto oben rechts) befindet sich der Bereich „Verknüpfte Konten".

Was bewirkt eine Verknüpfung?

Nach der Verknüpfung zweier Konten sieht jeder Nutzer automatisch alle Rezepte des anderen in seiner Sammlung – als wären es eigene. Rezepte des verknüpften Kontos sind in der Übersicht mit dem Namen des Besitzers gekennzeichnet. Die Verknüpfung gilt in beide Richtungen.

⚠️ Über verknüpfte Konten sichtbare Rezepte sind grundsätzlich nur lesbar. Bearbeitungsrecht erteilt man über die direkte Rezept-Freigabe (siehe oben).
Verknüpfung anfragen – Nutzer bereits registriert
  • Im Profil die E-Mail-Adresse des anderen Nutzers eingeben
  • Auf Anfragen tippen
  • Der andere Nutzer sieht im Profil unter „Offene Anfragen" eine Benachrichtigung (roter Zähler auf dem Profilfoto)
  • Nach dem Annehmen ist die Verknüpfung aktiv
Verknüpfung anfragen – Nutzer noch nicht registriert

Wenn die eingegebene E-Mail-Adresse noch kein Konto hat, wird automatisch eine Einladungs-E-Mail verschickt (gültig 48 Stunden).

  • E-Mail-Adresse eingeben und auf Anfragen tippen
  • Der Eingeladene erhält eine E-Mail mit einem persönlichen Link
  • Link öffnen → Einladungsseite zeigt wer einlädt
  • Auf Mit Google anmelden tippen (oder, falls bereits angemeldet, Einladung annehmen)
  • Die Verknüpfung wird sofort aktiviert
💡 Gesendete Einladungen werden im Profil unter „Gesendete Einladungen" angezeigt und können dort zurückgezogen werden.
Offene Anfragen annehmen oder ablehnen

Eingehende Anfragen erscheinen im Profil unter „Offene Anfragen" mit Annehmen und Ablehnen-Buttons. Ein roter Zähler auf dem Profilfoto signalisiert wartende Anfragen.

Verknüpfung aufheben

Im Profil das Papierkorb-Symbol neben der Verknüpfung antippen. Danach sind die Rezepte des anderen Nutzers sofort ausgeblendet.

Einstellungen

Ganz unten auf der Profilseite befindet sich der Einstellungsbereich. Die Einstellungen werden lokal auf dem Gerät gespeichert.

🖥️ Display beim Kochen nicht ausschalten

Standard: aktiviert. Wenn der Kochmodus aktiv ist, verhindert diese Einstellung, dass der Bildschirm nach kurzer Inaktivität automatisch sperrt. Praktisch beim Kochen nach Rezept, ohne das Gerät ständig zu entsperren.

Technisch: Screen Wake Lock API (Chrome 84+, Safari 16.4+, Firefox 126+). Ältere Browser ignorieren diese Einstellung stillschweigend.

Smartphone-Kurzbefehle

Mit einem eingerichteten Kurzbefehl lässt sich jedes Rezept direkt aus dem Browser-Teilen-Menü in die Sammlung importieren – ohne die App zu öffnen. Diese Methode funktioniert auch ohne installierte PWA und auf iOS.

Voraussetzung: Ein persönlicher API-Schlüssel. Dieser kann bei stephan@familie-ebert.net angefragt werden.

🍎 iPhone & iPad – iOS-Kurzbefehle

  • Die App Kurzbefehle öffnen → oben rechts auf + tippen → neuer Kurzbefehl
  • Aktion 1: „Eingabe aus Teilen-Menü empfangen" hinzufügen (Typ auf URLs setzen)
  • Aktion 2: „Web-Anfrage" hinzufügen:
    • Methode: POST
    • URL: https://rezepte.familie-ebert.net/api/import
    • Header: x-api-key → dein API-Schlüssel
    • Header: Content-Type → application/json
    • Body: {"url":"[Eingabe aus Schritt 1]"}
  • (Optional) Aktion 3: „Mitteilung anzeigen" → „Rezept gespeichert ✓"
  • Kurzbefehl benennen (z. B. „Rezept speichern") und im Teilen-Menü aktivieren
💡 Den Kurzbefehl testen: Rezeptseite im Safari öffnen → Teilen → Kurzbefehl antippen → nach wenigen Sekunden erscheint das Rezept in der App.

🤖 Android – HTTP Shortcuts

  • App „HTTP Shortcuts" (kostenlos von Waboodoo) aus dem Play Store installieren
  • Neuen Shortcut anlegen:
    • Methode: POST
    • URL: https://rezepte.familie-ebert.net/api/import
  • Unter Header hinzufügen: 
    Content-Type: application/json
x-api-key: <dein-schlüssel>
  • Unter Body (Custom Text): 
    {"url":"{share_text}"}
  • Shortcut speichern und über Teilen → HTTP Shortcuts → Rezept speichern testen
🔌

API-Zugang

Für den Import über Kurzbefehle steht ein einfacher API-Endpunkt bereit. Der Zugang ist mit einem persönlichen API-Schlüssel gesichert.

Rezept importieren
POST https://rezepte.familie-ebert.net/api/import

Headers:
  x-api-key: <dein-schlüssel>
  Content-Type: application/json

Body:
  { "url": "https://www.chefkoch.de/..." }

Antwort (201):
  { "id": 42, "message": "Rezept erfolgreich importiert" }

Antwort bei Duplikat (409):
  { "error": "Rezept bereits vorhanden", "existingId": 42 }
API-Schlüssel anfragen: stephan@familie-ebert.net
